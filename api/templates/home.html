LinkedIn Ad Library Viewer

Visualiza anuncios publicados en LinkedIn utilizando la API de LinkedIn, Flask y Python.

Iniciar sesión con LinkedIn

¿De qué se trata este proyecto?

Este proyecto es una aplicación web que utiliza varias tecnologías modernas para permitir a los usuarios ver datos de anuncios publicados en LinkedIn. A través de la autenticación OAuth, los usuarios pueden iniciar sesión con LinkedIn, y luego acceder a la biblioteca de anuncios usando la API de LinkedIn Ad Library.

Tecnologías utilizadas

Python

Python es el lenguaje de programación principal utilizado en este proyecto. La aplicación está construida con Flask, un micro-framework de Python para aplicaciones web.

Flask

Flask se utiliza para manejar las rutas, autenticación y la lógica backend. Permite la integración con la API de LinkedIn y procesa las solicitudes de los usuarios.

API de LinkedIn

La API de LinkedIn Ad Library permite acceder a datos de anuncios publicados en LinkedIn. A través de OAuth, los usuarios pueden autenticar su cuenta y acceder a esta información.

Next.js

Next.js es un framework de React que se utiliza en la parte del frontend para manejar el contenido dinámico y las interfaces del usuario de forma rápida y eficiente.

OAuth 2.0

El protocolo de autenticación OAuth 2.0 se usa para permitir a los usuarios iniciar sesión en LinkedIn de manera segura, proporcionando un token de acceso para interactuar con la API.

Flujo de trabajo del proyecto

  1. El usuario hace clic en el botón "Iniciar sesión con LinkedIn".
  2. Se redirige al usuario a la página de autenticación de LinkedIn a través de OAuth 2.0.
  3. Después de autenticarse, el usuario es redirigido de vuelta a la aplicación con un token de acceso.
  4. El backend en Flask utiliza el token de acceso para hacer solicitudes a la API de LinkedIn y recuperar los datos de los anuncios.
  5. Los datos de los anuncios se muestran al usuario en la interfaz web.

¿Cómo funciona la autenticación con LinkedIn?

Este proyecto utiliza OAuth 2.0 para la autenticación. Los usuarios son redirigidos a la página de LinkedIn, donde deben otorgar permisos para acceder a la información de su cuenta. Una vez autenticado, LinkedIn devuelve un token de acceso que se utiliza para realizar solicitudes autenticadas a la API de LinkedIn.

Cómo utilizar la aplicación

Para comenzar, haz clic en "Iniciar sesión con LinkedIn". Una vez autenticado, podrás visualizar los anuncios publicados en LinkedIn.
